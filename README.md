# BharatVid-Translator
English to Indian Language Video Translator automatically converts English video audio into Indian languages and merges it back. Supports Hindi, Tamil, Bengali, Telugu, Marathi, and more. Uses Python, SpeechRecognition, Google Translator, and gTTS for seamless multilingual video creation
English to Indian Language Video Translator

A Python project that automatically translates the audio of English videos into Indian regional languages and merges the translated audio back into the video. Ideal for content creators, educators, and anyone looking to make videos accessible to a wider audience.

Features

Extracts audio from video using FFmpeg

Converts audio to WAV for transcription

Transcribes English audio with SpeechRecognition

Translates text to Indian languages using deep-translator

Converts translated text to speech with gTTS

Replaces original video audio with translated speech

Supports multiple languages: Hindi, Tamil, Bengali, Telugu, Marathi, Gujarati, Kannada, Malayalam, Punjabi, etc.

Easy to change target language via TARGET_LANG
